# YouTube Trending Video Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on YouTube trending video datasets using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Features
- Data Cleaning
- Trend Analysis
- Likes vs Views Analysis
- Correlation Heatmap
- Channel Performance Analysis

## Dataset
Kaggle YouTube Trending Dataset

## Project Output
- Top trending channels
- Viewer engagement analysis
- Correlation visualizations
- Trending insights
